## COURSE DESCRIPTION
This course introduces fundamental machine learning concepts with a focus on Python programming, PyTorch deep learning, and AI agent workflows. Students will learn supervised and unsupervised learning, neural networks, model training pipelines, and how to leverage AI agents for tasks such as data preprocessing, model selection, code generation, testing, and documentation. Emphasis is on hands-on implementation using modern tools.

## PREREQUISITES
* Basic Python programming
* Linear algebra (matrices, vectors)
* Probability and statistics fundamentals

## COURSE OBJECTIVES
By the end of the course, students will be able to:
* Implement ML algorithms in Python
* Train and evaluate deep learning models using PyTorch
* Apply ML workflows to real-world datasets
* Build and integrate AI agents to assist in ML development and experimentation
* Evaluate model performance using appropriate metrics
* Demonstrate ethical awareness in AI/ML systems

## COURSE TOPICS (WEEKLY OUTLINE)
Week 1: Introduction to ML, Python review, tools setup  
Week 2: Data preprocessing, feature engineering  
Week 3: Supervised learning: regression, classification  
Week 4: Model evaluation, metrics, cross-validation  
Week 5: Unsupervised learning: clustering, dimensionality reduction  
Week 6: Intro to neural networks, backpropagation  
Week 7: PyTorch fundamentals, tensors, autograd  
Week 8: Deep learning training pipelines in PyTorch  
Week 9: CNNs and image classification  
Week 10: RNNs, sequence modeling (intro)  
Week 11: AI agents: prompt engineering and agent workflows  
Week 12: AI agents for dataset generation, augmentation, documentation  
Week 13: AI agents for debugging, testing, and evaluation automation  
Week 14: Final project development and presentations  
Week 15: Review and final assessment

## LABORATORY COMPONENT

* Building AI-assisted ML pipelines using agent tools
